Help and information

General info

This application provides an overview of all movies, one-shots and shows created for the Marvel Cinematic Universe (MCU).
In addition, a timeline with the optimal viewing order is supplied.
All titles, images and descriptions were taken from the Internet Movie Database.
By clicking the IMDb logo, the full webpage is displayed.
The application has a built-in browser, but it is possible to start the default browser by changing this in the settings.

Playing media

Every movie, one-shot and episode of a show has a play button.
It is possible to directly start playing the video by clicking this button.
However, the video files are not included in this application.
Owners of the properly purchased materials may place their video files in the proper media directory to enable viewing.
The application has a built-in media player, but it is possible to start the default media player by changing this in the settings.

F.A.Q.

How can I quickly find a certain movie or episode?
The search functionality provides a way to quickly search for any term.
All movies, one-shots, episodes and shows that have the term either in their title or in their description, will be returned.
The search results have a button that lets you jump right to its location in the overview.
Note: You can only jump to the general overview, not to the timeline.
What about future releases?
Certain data about the future is already known and has been included in the application.
By checking the "Include future releases" option in the settings, all known data will be displayed in the overview.
The collection is not up-to-date, what now?
The collection is loaded from a simple XML document. To get the latest data, it suffices to download the latest collection file.
In case the latest collection file is still outdated, you may update it yourself.
The file "collection.xml" can be found in the installation folder.
Beware: If the collection file is corrupted, the application will no longer work. Only change it if you know what you're doing!
My video won't play. What can I do?
First of all, doublecheck that the video file is in the correct place and has the right name.
It should be exactly the same (with the exception of the extension) as the image for the movie / one-shot / episode.
If everything looks fine, you may be trying to play an unsupported format.
Supported extensions are: .mkv, .mp4, .avi, .mpg, .mpeg, .mov, .wmv.
Why won't the embedded media player slider update while playing a video?
This is a known issue / bug. The media player is not entirely finished yet.
You are most likely better off using an external video player anyway.
If you truly wish to use the embedded player, look for an updated version of this App (it may have been fixed).
